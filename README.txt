 MovieHunt : Fetch List of movies from https://movies-sample.herokuapp.com/api/movies, and dsiplay  to user with search functionality and caching mechanism for offline support

   - Used Dagger for Dependency Injection
   - Used Retrofit to consume Json data
   - Used Moshi  to serialize JSON to Java objects
   - Used RxJava to setup asynchronous data streams
   - Used Picasso for displaying images
   - Implemented MVVM architecture with lifecycle aware liveData and Viewmodel components
   - RecyclerView to display Movies
   - MockK for Unit Testing
   - Room for Offline Support
